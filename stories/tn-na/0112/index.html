Ke kgona go taboga ka ona.

I can run with it.

Ke kgona go tlola ka ona.

I can jump with it.

Ke kgona go bina ka ona.

I can dance with it.

Ke kgona go thuma ka ona.

I can swim with it.

Ke kgona go tlola kgole ka ona.

I can skip with it.

Ke kgona go raga ka ona.

I can kick with it.

Ke kgona go tshaba ka ona.

I can escape with it.

Mme ga nkake ka fofa ka ona.

But I can never fly with it.

Written by: Clare Verbeek, Thembani Dladla, Zanele Buthelezi
Illustrated by: Mlungisi Dlamini, Ingrid Schechter
Translated by: Domitilla Naledi Madi
Language: Setswana
Level: Level 1
Source: My body from African Storybook
